## where 条件查询

### 级别1: 简单条件

```javascript
// 等于
db.collection('user').where('age == 18').get()

// 不等于
db.collection('user').where('age != 18').get()

// 大于
db.collection('user').where('age > 18').get()

// 大于等于
db.collection('user').where('age >= 18').get()

// 小于
db.collection('user').where('age < 18').get()

// 小于等于
db.collection('user').where('age <= 18').get()
```

### 级别2: 多条件 AND

```javascript
// 两个条件都要满足
db.collection('user').where('age > 18 && status == 1').get()

// 多个条件
db.collection('user').where('age > 18 && status == 1 && vip == true').get()
```

### 级别3: 多条件 OR

```javascript
// 满足其中一个条件即可
db.collection('user').where('age > 18 || vip == true').get()

// 复杂组合
db.collection('user').where('(age > 18 && vip == true) || score > 100').get()
```

### 级别4: in 查询

```javascript
// 字段值在数组中
db.collection('user').where('status in ["active","pending"]').get()

// 字段值不在数组中
db.collection('user').where('!(status in ["deleted","banned"])').get()
```

### 使用云端环境变量

```javascript
// $cloudEnv_uid - 当前用户ID
db.collection('user').where('uid == $cloudEnv_uid').get()

// $cloudEnv_now - 服务器时间戳
db.collection('user').where('create_time < $cloudEnv_now').get()

// $cloudEnv_clientIP - 当前客户端IP
db.collection('user').where('ip == $cloudEnv_clientIP').get()
```

## field 字段过滤

### 级别1: 指定字段

```javascript
// 返回 _id, name, age 字段
db.collection('user').field('name,age').get()
```

### 级别2: 嵌套字段

```javascript
// 返回嵌套对象的字段
db.collection('user').field('user.name,user.age').get()
```

### 级别3: 字段别名

```javascript
// 重命名字段
db.collection('user').field('name as userName, age as userAge').get()

// 返回结果
{
  "_id": "xxx",
  "userName": "张三",
  "userAge": 18
}
```

### 级别4: 字段运算

```javascript
// 对字段进行运算
db.collection('score').field('add(score1, score2) as totalScore').get()

// 使用数据库运算方法
db.collection('class').field('eq(grade,6) as isTopGrade').get()
```

## orderBy 排序

### 级别1: 单字段排序

```javascript
// 升序(asc 可以省略)
db.collection('user').orderBy('age asc').get()
db.collection('user').orderBy('age').get()

// 降序
db.collection('user').orderBy('age desc').get()
```

### 级别2: 多字段排序

```javascript
// 先按 age 升序,age 相同再按 score 降序
db.collection('user').orderBy('age asc, score desc').get()

// 简写形式
db.collection('user').orderBy('age, score desc').get()
```

## 分页查询

### 标准分页模式

```javascript
async function getPageData(pageIndex = 1, pageSize = 20) {
  const db = uniCloud.databaseForJQL()
  const skip = (pageIndex - 1) * pageSize;

  const res = await db.collection('user')
    .where('status == 1')
    .orderBy('create_time desc')
    .skip(skip)
    .limit(pageSize)
    .get({ getCount: true }); // 同时获取总数

  return {
    list: res.data,
    total: res.count,
    pageIndex,
    pageSize
  };
}

// 使用
const result = await getPageData(1, 20);
console.log(result.list);   // 数据列表
console.log(result.total);  // 总记录数
```

### 注意事项

- limit 默认值是 100
- limit 最大值为 1000(阿里云和腾讯云)
- 使用 skip + limit 进行分页
- 使用 `get({ getCount: true })` 同时获取总数

## 复杂查询条件

### 比较数据库字段

```javascript
// 查询数学得分比语文高的记录
db.collection('score').where('math > chinese').get()

// 查询两个字段相等的记录
db.collection('user').where('field1 == field2').get()
```

### 使用数据库运算方法

```javascript
// 查询语文数学总分大于150的数据
db.collection('score').where('add(chinese,math) > 150').get()

// 查询日期字段
db.collection('test').where('deadline < new Date().getTime()').get()
```

## 正则查询

### 搜索场景

```javascript
// 场景: 商品搜索
const searchVal = '手机';
const res = await db.collection('goods')
  .where(`${new RegExp(searchVal, 'i')}.test(name)`)
  .get();
```

### 在 unicloud-db 组件中使用

```javascript
export default {
  data() {
    return {
      searchVal: ''
    }
  },
  computed: {
    where() {
      return `${new RegExp(this.searchVal, 'i')}.test(name)`
    }
  }
}
```

## 统计查询

### 场景1: 纯计数

```javascript
const count = await db.collection('user')
  .where('age > 18')
  .count();

// 返回: { "total": 100 }
```

### 场景2: 查询同时计数

```javascript
const res = await db.collection('user')
  .where('age > 18')
  .get({ getCount: true });

// res.data 是数据列表
// res.count 是总数
```

### 场景3: 只查一条记录

```javascript
const res = await db.collection('user')
  .where('name == "张三"')
  .get({ getOne: true });

// 返回单条记录
res.data = {
  "_id": "xxx",
  "name": "张三"
}
```

## 返回值说明

### 正常请求返回结果

**查询数据:**
```javascript
{
  errCode: 0,
  errMsg: '',
  data: []
}
```

**批量发送数据库查询请求:**
```javascript
{
  errCode: 0,
  errMsg: '',
  dataList: [] // dataList内每一项都是一个查询数据的响应结果
}
```

**新增数据(单条):**
```javascript
{
  errCode: 0,
  errMsg: '',
  id: '' // 新增数据的id
}
```

**新增数据(多条):**
```javascript
{
  errCode: 0,
  errMsg: '',
  ids: [], // 新增数据的id列表
  inserted: 3 // 新增成功的条数
}
```

**删除数据:**
```javascript
{
  errCode: 0,
  errMsg: '',
  deleted: 1 // 删除的条数
}
```

**更新数据:**
```javascript
{
  errCode: 0,
  errMsg: '',
  updated: 1 // 更新的条数，数据更新前后无变化则更新条数为0
}
```

### 请求报错返回格式

```javascript
{
  errCode: "", // 错误码
  errMsg: "", // 错误信息
}
```

**常见错误码:**

| 错误码 | 说明 |
|--------|------|
| TOKEN_INVALID_INVALID_CLIENTID | token校验未通过（设备特征校验未通过） |
| TOKEN_INVALID | token校验未通过（云端已不包含此token） |
| TOKEN_INVALID_TOKEN_EXPIRED | token校验未通过（token已过期） |
| TOKEN_INVALID_WRONG_TOKEN | token校验未通过（token校验未通过） |
| SYNTAX_ERROR | 语法错误 |
| PERMISSION_ERROR | 权限校验未通过 |
| VALIDATION_ERROR | 数据格式未通过 |
| DUPLICATE_KEY | 索引冲突 |
| SYSTEM_ERROR | 系统错误 |
