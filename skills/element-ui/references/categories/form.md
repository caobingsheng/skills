# Form 表单组件

表单组件提供了完整的表单输入和验证功能。

## 组件列表

### 输入组件
- **Radio 单选框**：单选框组件
- **Checkbox 多选框**：多选框组件
- **Input 输入框**：输入框组件
- **InputNumber 计数器**：计数器组件
- **Select 选择器**：选择器组件
- **Cascader 级联选择器**：级联选择器
- **Switch 开关**：开关组件
- **Slider 滑块**：滑块组件
- **TimePicker 时间选择器**：时间选择器
- **DatePicker 日期选择器**：日期选择器
- **DateTimePicker 日期时间选择器**：日期时间选择器
- **Upload 上传**：上传组件
- **Rate 评分**：评分组件
- **ColorPicker 颜色选择器**：颜色选择器
- **Transfer 穿梭框**：穿梭框组件

### 表单容器
- **Form 表单**：表单组件，提供表单验证

## 何时使用

- **Input**：需要文本输入时
- **Select**：需要从选项中选择时
- **Form**：需要表单验证时
- **Upload**：需要文件上传时

## 相关组件

- Basic 组件：Button、Icon
- Notice 组件：Alert、Message
