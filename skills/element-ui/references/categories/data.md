# Data 数据展示

数据展示组件用于展示各种类型的数据。

## 组件列表

### 表格和列表
- **Table 表格**：表格组件
- **Tree 树形控件**：树形控件
- **Descriptions 描述列表**：描述列表组件

### 标记和标签
- **Tag 标签**：标签组件
- **Badge 标记**：标记组件
- **Avatar 头像**：头像组件

### 进度和状态
- **Progress 进度条**：进度条组件
- **Skeleton 骨架屏**：骨架屏组件
- **Empty 空状态**：空状态组件
- **Result 结果**：结果组件
- **Statistic 统计数值**：统计数值组件

### 分页
- **Pagination 分页**：分页组件

### 其他
- **Calendar 日历**：日历组件
- **Image 图片**：图片组件
- **Backtop 回到顶部**：回到顶部组件
- **InfiniteScroll 无限滚动**：无限滚动组件

## 何时使用

- **Table**：需要展示表格数据时
- **Tree**：需要展示树形数据时
- **Pagination**：需要分页时
- **Tag**：需要标签标记时

## 相关组件

- Form 组件：表单数据
- Notice 组件：Alert、Message
