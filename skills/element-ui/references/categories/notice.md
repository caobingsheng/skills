# Notice 通知

通知组件用于向用户展示各种提示和反馈信息。

## 组件列表

### 静态提示
- **Alert 警告**：警告组件

### 动态提示
- **Loading 加载**：加载组件
- **Message 消息提示**：消息提示组件
- **Notification 通知**：通知组件

### 交互提示
- **MessageBox 弹框**：弹框组件

## 何时使用

- **Alert**：需要静态警告提示时
- **Message**：需要轻量级消息提示时
- **Notification**：需要右侧通知时
- **MessageBox**：需要用户确认时
- **Loading**：需要加载状态时

## 相关组件

- Others 组件：Dialog、Drawer
- Form 组件：表单验证反馈
