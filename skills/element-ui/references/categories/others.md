# Others 其他

其他组件提供了各种常用的交互和展示功能。

## 组件列表

### 弹窗和提示
- **Dialog 对话框**：对话框组件
- **Drawer 抽屉**：抽屉组件
- **Tooltip 文字提示**：文字提示组件
- **Popover 弹出框**：弹出框组件
- **Popconfirm 气泡确认框**：气泡确认框组件

### 卡片和容器
- **Card 卡片**：卡片组件
- **Collapse 折叠面板**：折叠面板组件

### 展示组件
- **Carousel 走马灯**：走马灯组件
- **Timeline 时间线**：时间线组件

### 辅助组件
- **Divider 分割线**：分割线组件

## 何时使用

- **Dialog**：需要对话框时
- **Drawer**：需要侧边抽屉时
- **Card**：需要卡片容器时
- **Tooltip**：需要提示时
- **Collapse**：需要折叠面板时

## 相关组件

- Notice 组件：MessageBox
- Navigation 组件：Dropdown
- Data 组件：各种展示组件
