# Navigation 导航

导航组件用于页面导航和流程引导。

## 组件列表

### 菜单导航
- **NavMenu 导航菜单**：导航菜单组件
- **Tabs 标签页**：标签页组件
- **Dropdown 下拉菜单**：下拉菜单组件

### 面包屑和页头
- **Breadcrumb 面包屑**：面包屑组件
- **PageHeader 页头**：页头组件

### 步骤条
- **Steps 步骤条**：步骤条组件

## 何时使用

- **Menu**：需要侧边导航时
- **Tabs**：需要标签页切换时
- **Breadcrumb**：需要面包屑导航时
- **Steps**：需要步骤引导时

## 相关组件

- Basic 组件：Icon、Button
- Others 组件：Dropdown
