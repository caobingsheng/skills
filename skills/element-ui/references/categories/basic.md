# Basic 基础组件

基础组件提供了页面布局、样式规范和基础交互元素。

## 组件列表

### 布局组件
- **Layout 布局**：栅格布局系统，基于 24 栅格
- **Container 布局容器**：容器布局组件，用于页面结构

### 样式规范
- **Color 色彩**：Element UI 色彩规范
- **Typography 字体**：字体排版规范
- **Border 边框**：边框样式规范
- **Icon 图标**：图标组件库

### 交互组件
- **Button 按钮**：基础按钮组件
- **Link 文字链接**：文字链接组件

## 何时使用

- **Layout**：需要栅格布局时
- **Button**：需要按钮交互时
- **Icon**：需要图标时

## 相关组件

- Form 组件：表单场景
- Navigation 组件：导航场景
